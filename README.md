# FinTrack

Aplicação web para controle de finanças pessoais: registro de receitas e despesas, metas, gráficos e relatórios. Interface single-page sem backend, com dados persistidos via localStorage.

## Funcionalidades

- Registro de receitas e despesas por categoria
- Transações recorrentes mensais, geradas automaticamente a cada mês
- Filtros por texto, tipo, categoria e mês
- Metas financeiras com barra de progresso e depósitos
- Resumo inteligente com análises e dicas automáticas
- Gráficos de despesas por categoria e de evolução mensal
- Exportação dos dados em CSV e PDF
- Tema claro e escuro

## Tecnologias

- HTML, CSS e JavaScript puro
- Chart.js (via CDN) para os gráficos
- localStorage para persistência dos dados

## Como usar

Abra o arquivo `index.html` diretamente no navegador. Nenhuma instalação necessária; os gráficos usam a Chart.js carregada via CDN, então é preciso conexão com a internet na primeira carga.

Para publicar online, basta ativar o GitHub Pages no repositório.
